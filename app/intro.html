Welcome to Landing. To use this extension, create a bookmark folder called landing in the Other bookmarks folder.

Inside the landing folder, each folder will be shown as a category of bookmarks on the New Tab page.

Finally, bookmarks directly inside these category folders will be shown as links on the New Tab page.

You can customize the visuals of the New Tab page by visiting the Options page.

The hierarchy should look something like this:

  • Other bookmarks
    • landing
      • Search
        • Google
        • DuckDuckGo
        • Wikipedia
      • Social
        • Twitter
        • Google+
        • Facebook
      • News
        • Reddit
        • BBC
        • CNN
